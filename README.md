# Tic-Tac-Toe
This project is Project-Based Learning, where I learn about how to code with Python.

## Introduction
This game is a Tic-Tac-Toe game. It is two player game; each player need to place thier owns mark (X,Y) on the board. Who ever get 3 mark stick to each other wins.

## Installation
1. Python version 3.10.8
2. random and numpy library
